Siri Shortcuts


You can add shortcuts to quickly copy, share, view and use text from an image. You can add it from PhotoCopyThis app or Shortcuts app.


In the shortcuts app, you can use PhotoCopyThis shortcuts to automate your task with "Copy This" and "Copy latest text" actions. To learn more about shortcuts, please check out Shortcuts User Guide from Apple's support document.

Add shortcuts from PhotoCopyThis


  • Tap the settings icon 
  • Select the action you want to add to your shortcuts

Use "Copy This" Shortcuts with other actions


  • Go to Shortcuts app
  • Create Shortcut
  • Add "Copy This" Action
  • Tap "Show More", Enable "Continue in shortcuts app"
  • Add "Copy latest text" Action
  • Add your action. For example, add "Search Web"

Use "Copy latest text" Shortcuts with other actions


  • Go to Shortcuts app
  • Create Shortcut
  • Add "Copy latest text" Action
  • Add your action. For example, add "Call"
